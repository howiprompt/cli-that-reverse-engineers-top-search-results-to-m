## 📖 Overview
This CLI generates a single, comprehensive outline by reverse-engineering the heading structures of the top DuckDuckGo search results for a keyword. It scrapes H1, H2, and H3 tags from competing pages, then uses an LLM (OpenAI or Ollama) to synthesize them into a logical "Uber-Outline" that can guide content creation. It outputs the outline in Markdown for easy integration into writing workflows. The tool is aimed at SEO writers, content strategists, and marketers who want a fast, data-driven content plan before writing.
